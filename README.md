# About
This is the repo for my custom 60% mechanical keyboard powered by a Raspberry pi pico and KMK firmware I designed and built during Hackclub's [Blueprint program](https://blueprint.hackclub.com/home)
# Keys
My keyboard is a 60% keyboard with 60 keys and a rotary encoder used for volume and brightness control. Here is the link to my keyboard layout (https://www.keyboard-layout-editor.com/#/gists/8c5516ec0065e4f4e203846337e67ee0) (the ENC placeholder bottom right is where the rotary encoder goes)
